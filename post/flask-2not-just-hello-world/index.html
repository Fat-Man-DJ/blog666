Flask 2:Not Just Hello World

Flask 2!NJHW!

终于有时间写博客了

Flask 2:

Jinja2

Jinja2是一个模板引擎

代码：

<code>
    {{ content }}
</code>

在Flask里使用Jinja2

在你的项目文件夹里创建一个叫templates的文件夹

mkdir templates

然后新建一个叫temp.html的文件

写入如下内容

<html>
    <head>
    </head>
    <body>
        <center>
            {{ name }},How Are You?
        </center>
    </body>
</html>

再在python文件里输入

from flask import  *
app = Flask(__name__)



@app.route('/')
def index():
    return render_template('temp.html',name='666')

app.run(port=80)

运行此文件后，

访问localhost

你可以看到666,How are you?

那就对了！
